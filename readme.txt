For DT Tree, compile the Java files inside an IDE.

The program is run by passing in the arguments: “trainingFilePath” “testFilePath” 

The entry point of the program is the Main class.
If no parameters are specified, then it will run the 10 training and test files (01..10) e.g. hepatitis-training-run01.dat and hepatitis-test-run01.dat
